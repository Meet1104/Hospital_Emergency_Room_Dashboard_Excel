# 🏥 Hospital Emergency Room Dashboard

An interactive **Hospital Emergency Room Dashboard** built using **Microsoft Excel** for analyzing patient flow, emergency cases, waiting times, admissions, and hospital performance metrics.

---

# 📊 Dashboard Overview

This dashboard helps in:

- Monitoring emergency room patient data
- Tracking admission statistics
- Analyzing patient wait times
- Visualizing department-wise performance
- Understanding daily/monthly hospital trends
- Improving hospital operational decisions

---

# 🚀 Features

✅ Interactive Dashboard  
✅ Dynamic Charts & Graphs  
✅ KPI Cards  
✅ Emergency Case Analysis  
✅ Patient Admission Tracking  
✅ Department Performance Insights  
✅ Excel-Based Automation  
✅ User-Friendly Design

---

# 🛠️ Tools & Technologies Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas
